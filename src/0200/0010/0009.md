### 排序

问题：在一个字符串数组中有红、黄、蓝三种颜色的球，且个数不相等、顺序不一致，请为该数组排序。使得排序后数组中球的顺序为:黄、红、蓝。

例如：红蓝蓝黄红黄蓝红红黄红，排序后为：黄黄黄红红红红红蓝蓝蓝。

```js
let arr = "红黄蓝蓝红黄红蓝黄红黄红黄".split("");
let obj = { 黄: 0, 红: 1, 蓝: 2 };
arr.sort((prev, next) => obj[prev] - obj[next]);
```
